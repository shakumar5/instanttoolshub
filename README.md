# ToolsHub

A collection of 10+ free, blazing-fast, 100% client-side developer utilities. No data ever leaves your browser.

## Tools

| Tool | Description |
|------|-------------|
| JSON Formatter & Validator | Format, minify, validate, and convert JSON data |
| Base64 Encoder & Decoder | Encode/decode text, images, and files |
| URL Encoder & Decoder | Percent encoding, query string parsing, UTM builder |
| RegEx Tester & AI Generator | Live regex testing with match highlighting |
| JWT Decoder & Debugger | Decode, verify, and inspect JSON Web Tokens |
| CSS/JS/HTML Minifier & Beautifier | Compress or format web code files |
| Markdown Editor & README Creator | Split-pane editor with live preview and templates |
| Color Converter & Palette Picker | HEX, RGB, HSL, CMYK conversion with gradient builder |
| Unix Timestamp Converter | Epoch conversion, timezone grid, cron parser |
| Hash & HMAC Generator | MD5, SHA-1, SHA-256, SHA-512 with HMAC support |

## Tech Stack

- [Astro](https://astro.build/) v6
- Vanilla JavaScript (no frameworks)
- Self-hosted fonts via `@fontsource`

## Getting Started

```sh
npm install
npm run dev
```

The dev server runs at `http://localhost:4321`.

## Commands

| Command | Action |
|---------|--------|
| `npm run dev` | Start local dev server |
| `npm run build` | Build production site to `./dist/` |
| `npm run preview` | Preview the production build locally |
